Для запуска программы запустите скрипт main.py !!!

Проект разбит на 2-е основые директории:
    1 - api.
    2 - database.

В "api" происходит взаимодействие с API HeadHunter. Прописанны классы, методы, функции необходимые для работы.
В "database" прописан весь функционал для работы с PostgresSQL (Запись и чтение данных).
